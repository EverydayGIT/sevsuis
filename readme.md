# О проекте
Данный проект делается студентами кафедры ИС. По окончанию планируется получить рабочий веб-сайт, призванный показать абитуриентам преимущества поступление на нашу кафедру.

- Bootsrap
- JavaScript
- jQuery
- Git
- Возможно что-то ещё.
